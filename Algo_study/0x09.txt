<BFS>
- Breadth First Search
- 다차원 배열에서 각 칸을 방문할 때 너비를 우선으로 방문하는 알고리즘
- 그래프라는 자료구조에서 모든 노드를 방문하기 위한 자료구조

[BFS의 흐름도]
1. 시작하는 칸을 큐에 넣고 방문했다는 표시를 남김.
2. 큐에서 원소를 꺼내어 그 칸에 상하좌우로 인접한 칸에 대해 3번을 진행.
3. 해당 칸을 이전에 방문했다면 아무 것도 하지 않고, 처음으로 방문했다면 방문했다는 표시를 남기고 해당 칸을 큐에 삽입함.
4. 큐가 빌 때까지 번호 2를 계속 반복, 모든 칸이 큐에 1번씩 들어가므로 시간복잡도는 칸이 N개 일때 O(N).
**BFS에서 그래프를 탐색하는 과정에서 좌표값을 x,y 나누어서 적는 것 보다. pair STL을 사용하는 것이 더 낫다.

이 STL은 <utility> 헤더파일 안에 선언되어 있다.

[BFS 기본 구조]
#include <bits/stdc++.h>
using namespace std;
#define X first
#define Y second
int board[502][502] = {};
bool vis[502][502];
int n = 7, m = 10;
int dx[4] = { -1,0,1,0 }; // 탐색순서: 상,우,하,좌
int dy[4] = { 0,1,0,-1 };

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	queue<pair<int, int>> Q; //queue 생성, pair를 이용해서 정수쌍으로 좌표값을 저장함
	vis[0][0] = 1; // 시작 좌표의 visit값 = 1
	Q.push({ 0,0 }); // 좌표값 in
	// BFS 시 무조건 탐색 후 범위 내부 확인 후 visit값=1 후 값 push

	while (!Q.empty()) {
		pair<int, int> cur = Q.front(); // 탐색할 좌표 cur에 할당
		Q.pop(); // 그 좌표는 버리고
		for (int i = 0; i < 3;i++) {
			int nx = cur.X + dx[i];
			int ny = cur.Y + dy[i];
			if (nx < 0 || nx>=7 || ny < 0 || ny>=10) continue; // 범위 7.10 벗어나면 skip
			if (vis[nx][ny] || board[nx][ny] != 1) continue; // 방문 O or board 값이 1이면 skip
			vis[nx][ny] = 1;
			Q.push({ nx,ny });
		}
	}

}

[BFS 구현시 주의할 점]
1. 시작점에 방문했다는 표시를 남기지 않는다.
2. 큐에 "넣을 때" 무조건 "방문했다는 표시"를 남겨야 한다. 큐에서 빼낼 때 방문했다는 표시를 남기면 안된다.
>> 이러면 한 노드를 여러번 방문하잖아! 런타임 에러 발생 가능.
3. 이웃한 원소가 범위를 벗어났는지에 대한 체크를 잘못했다.

<tuple과 tie>
거의 안쓰이는 것 같으나... 결국 코딩을 하는 과정에서 배열이 최대 3차원까지 왠만하면 나올 텐데 이럴 때 이 두개를 모르면 진짜 애먹을 듯. 세개의 변수를 처리해 주는 과정에서 큐를 여러개 만들어야 하니 3차원이면 최소 2개, 많게는 3개가 필요하고 이러면 메모리도 더 잡아먹을 거고, 소요시간도 많이 들게 된다. 그러니 이정도는 꼭 알고 가는게 좋을 것 같다.

[tuple]
header: <tuple.h>
선언: tuple<int, double, string> tp; 이런식으로 pair와 동일하게 선언해주면 됨.
초기화 or 입력: tuple<int, double, string> tp = {2,2.5,"hello"};
출력: get<0 여기에는 출력할 변수의 인덱스 입력>(tp 여기에는 출력할 튜플이 담긴 변수 입력)

[tie]
tie(각 변수의 타입과 일치하는 변수들을 입력) = 튜플명 입력 >> 이걸 통해 튜플의 각 항을 변수별로 분리할 수 있음.

ex) int first;
     double second;
     string third;

     tie(first, second, third) = tp;

ex) tie(X, Y, Z) = Q.front();

이런식으로. 
pair는 tie와 같이 분리하고 싶다면 auto 연산자를 사용해서 분리하면 된다.
