<백트래킹>
정의: 현재 상태에서 가능한 모든 후보군을 따라 들어가며 탐색하는 알고리즘

>> 탐색이라 뭔가 bfs나 dfs 같을 수 있을지 모르지만 거리 혹은 조합 형태의 탐색이 아니라 순열 형태의 탐색이면 백트래킹을 사용해서 탐색을 진행할 수 있는 것 같다. 그리고 백트래킹의 경우 전체의 경우의 수를 전부 탐색한다고 가정할 때 사용하는듯..

[순열과 조합]
수를 순서 상관있이 뽑는 경우 순열을 사용
순서 상관없이 뽑는 경우 조합을 사용
이건 당연하다.
순열과 조합을 사용하는 과정에서 현재까지는 직접 코드를 통해 뽑고 안뽑고를 결정했는데 이걸 코드 한줄로 대체할 수 있는 함수가 있다.

>> [next_permutation] 이거다. **permutation: 순열**

[next_permutation 순열의 예시]
int a[3] = {1,2,3};
do{
	for(int i = 0; i < 3; i++)
	  cout << a[i];
	cout << '\n';
}while(next_permutation(a, a+3));

>> 이걸 이용하면 a배열의 오름차순으로 정렬된 순열을 모두 얻을 수 있다.
     next_permutation에 인자로 전달되는 배열은 항상 오름차순으로 정렬되어 있어야 한다.
     중복 원소들은 중복을 제외하고 순열을 생성해준다.
>> if 내림차순의 순열을 구하고 싶다면 prev_permutiation 함수를 이용하면 된다.

[prev_permutation 조합의 예시]
int a[4] = {0,0,1,1};
int num[4] = {1,2,3,4};

do{
	for(int i = 0; i < 3; i++){
	  if(a[i] == 1)
	   cout << num[i] << ' ';
	}
	cout << '\n';
   }while(prev_permutation(a,a+4);

>> 이러면 오름차순의 a배열의 조합을 얻을 수 있다.
