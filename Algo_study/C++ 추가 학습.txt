<C++ 추가 학습>
[범위 지정 연산자] >> ::
범위 지정 연산자는 말 그대로 범위를 지정하는 연산자임.
예를들어 변수, 함수, 열거형 등에 사용해서 특정 변수, 함수, 열거형을 어디에 선언된 것을 사용할 것인지를 정해주는 것.
::변수명 >> 이렇게 범위 지정 연산자 앞에 아무것도 적지 않을 시 전역으로 선언된 변수 및 함수를 가져다 사용한다는 것.
클래스명:: 식별자 >> 특정 클래스 안의 식별자를 사용한다는 것.
네임스페이스명::식별자 >> 특정 네임스페이스 안의 식별자를 사용한다는 것이다.

C++ 에는 다른 클래스 혹은 네임스페이스 안에 같은 함수명을 갖는 함수들이 존재할 수 있는데 이 때 범위 지정 연산자를 이용하여 특정 클래스 혹은 네임스페이스를 정하여 구분하여 사용할 수 있다는 장점이 있다.


[STL(standard template library)]
- 주요 구성
 1) 컨테이너(container)
  데이터를 저장하고 관리하는 클래스 템플릿, 다양한 데이터 구조를 제공하여 데이터를 저장하고 조작할 수 있게 돕는다.
  ex) vector, list, queue, deque, set, stack, map 등...
 2) 반복자(Iterator)
  컨테이너의 요소를 순회하거나 접근하는데 사용되는 객체이다. 포인터와 유사한 기능을 한다고 생각하면 된다.
  종류: 입력 반복자(데이터 읽음), 출력 반복자(데이터 씀), 순방향 반복자(읽기와 쓰기가 가능(요소를 한 방향으로 읽음)),
         양방향 반복자(요소를 역방향으로 읽음), 임의 접근 반복자(양방향 반복자 기능 포함, 임의의 위치로 이동할 수 있음)
  ex) begin(), end()
 3) 알고리즘(Algorithm)
  데이터를 처리하고 조작하는 함수들의 집합. <algorithm.h> 헤더파일에 정의되어 있다.
  ex) sort(), reverse(), count()... 등
